# 版式设计

归根结底，版式设计就是对「点、线、面」的排布组合。

## 三要素

- 点：聚焦，强调
- 线：骨骼，支撑，区域划分
- 面：背景，环境

## 形式原理

- 重复交错
- 节奏韵律
- 对称平衡
- 对比调和
- 比例适度
- 变异秩序
- 虚实留白
- 变化统一

## 禁忌

- 避头尾法则：部分标点不能出现在行首，例如 ，、；：。？“” 《》
- 单字不成行
- 标题不落底
- 删除文字间的多余空格
- 段首
